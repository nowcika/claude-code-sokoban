？ 게임 설명 & Claude Code 배우기

이 게임의 각 방은 Claude Code가 실제로 사용하는 설정 파일/디렉토리입니다. 바닥에 흩어진 짐(📦)은 그 설정에 들어가는 항목이고요. 모든 짐을 자기 방의 목표 칸(점선)으로 밀어 넣으면 클리어!

  • settings.json — 훅 🪝 · 권한 🔐 · 환경변수 🌱 · 모델 🧠 설정
  • CLAUDE.md — 프로젝트 메모리 📌 · 코딩 규칙 📐 (항상 컨텍스트에 로드)
  • .mcp.json — MCP 서버 🔌 연결 정의
  • .claude/commands/ — 커스텀 슬래시 명령 ⚡ (/명령)
  • .claude/agents/ — 서브에이전트 🤖 정의
  • .claude/skills/ — 스킬 🎯 (SKILL.md)
  • settings.local.json — 개인용 로컬 오버라이드 🔒 (gitignore 대상)
  • .gitignore — 무시 규칙 🚫

짐을 정확한 방에 넣어야 인정됩니다. 예를 들어 훅 🪝 짐은 반드시 settings.json 방의 목표 칸으로 가야 해요. 최소 이동 수로 클리어하는 게 목표입니다.
